DNS (Domain Name Server)

A Domain Name Server (DNS) is a server that translates human-readable domain names into numerical IP addresses that computers use to identify each other on the network. Without DNS, you would have to memorize a string of numbers like 142.250.80.46 just to visit Google instead of simply typing google.com.

DNS works like the internet's phone book. When you type a website address, your browser asks the DNS server "what is the IP address for this domain?" and the DNS server responds with the correct address so your browser knows where to send the request.

💡 Did you know? The DNS system handles over 3.5 trillion queries every single day worldwide — making it one of the most used systems on the internet!
