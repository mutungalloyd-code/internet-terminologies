Web Browser

A Web Browser is a software application used to access, retrieve, and display information on the World Wide Web. When you type a URL into a browser, it sends a request to a web server using HTTP, receives the response, and renders the page for you to see and interact with.

Popular browsers include Google Chrome, Mozilla Firefox, Microsoft Edge, and Safari. All browsers do the same core job — interpreting HTML, CSS, and JavaScript to display web pages — but they may render things slightly differently.

💡 Did you know? The world's first web browser was called WorldWideWeb (later renamed Nexus), created by Tim Berners-Lee in 1990. It looked nothing like the browsers we use today!
