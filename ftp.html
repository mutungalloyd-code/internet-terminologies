FTP (File Transfer Protocol)

FTP (File Transfer Protocol) is a network protocol used to transfer computer files between a client and a server over a TCP/IP network. It is one of the oldest internet protocols still in use today, originally developed in the 1970s. FTP allows users to upload files to a server or download files from one.

FTP works using two separate channels — a command channel for sending instructions and a data channel for transferring files. Web developers commonly use FTP to upload website files to a hosting server. A more secure version called SFTP (Secure FTP) encrypts the data during transfer to protect it from interception.

💡 Did you know? FTP was one of the first internet protocols ever created, predating the World Wide Web by nearly 20 years!
