Internet Terminology

Your go-to guide for understanding the key concepts that power the modern web. Explore 16 essential internet terms explained clearly and simply.

Explore All Terms

🌐
World Wide Web
The global system of interlinked hypertext documents accessed via the internet.
📡
TCP/IP
The communication protocol that defines how data is transmitted over the internet.
🖥️
Web Browser
Software used to access, retrieve and display information on the Web.
🗄️
Web Server
Hardware and software that stores, processes and delivers website content.
🔗
URL
A web address that defines the location of a resource on the internet.
📖
DNS
Translates human-readable domain names into numerical IP addresses.
🔄
HTTP
The set of rules for exchanging data on the World Wide Web.
🏢
Intranet
A private internal network restricted to an organization's employees.
🤝
Extranet
A controlled network giving authorized external partners limited intranet access.
🏗️
Multitier Architecture
A design pattern separating application functionality into distinct layers.
📁
FTP
A protocol used to transfer files between a client and server over a network.
📝
HTML
The standard markup language used to create and structure web pages.
1️⃣
Web 1.0
The first era of the Web — static pages, read-only, limited interaction.
2️⃣
Web 2.0
The interactive, social web where users create and share content.
3️⃣
Web 3.0
The decentralized, blockchain-based next generation of the internet.
4️⃣
Web 4.0
The intelligent web integrating humans, machines and the physical world.
