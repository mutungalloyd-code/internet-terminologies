HTTP (HyperText Transfer Protocol)

HTTP (HyperText Transfer Protocol) is a set of rules used for exchanging data on the World Wide Web, enabling browsers to request and receive web content from servers. It follows a simple request-response cycle: the client sends a request message containing the method, path, headers and body; the server receives it, decides what to do, then sends back a response containing a status code, headers and body.

HTTPS is the secure version of HTTP — the "S" stands for Secure, meaning all data is encrypted before being sent. You can tell a site uses HTTPS by the padlock icon in your browser's address bar. Today, most websites use HTTPS to protect users' data from being intercepted.

💡 Did you know? The most common HTTP status codes are 200 (success), 404 (page not found), and 500 (server error). You've probably seen a 404 page before!
