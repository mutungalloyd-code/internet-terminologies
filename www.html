World Wide Web (WWW)

The World Wide Web (WWW) is a global information system of interlinked hypertext documents and multimedia accessed via the internet using HTTP and URLs.

It was invented by Tim Berners-Lee in 1989. The Web is not the same as the internet — the internet is the network itself, while the Web is a service that runs on top of it.

💡 Did you know? The first ever website went live on August 6, 1991 and it simply explained what the World Wide Web was!
