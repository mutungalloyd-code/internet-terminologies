URL (Uniform Resource Locator)

A URL (Uniform Resource Locator) is a URI that acts like a web address, defining the location and retrieval mechanism for resources on the internet. Every page, image, and file on the web has a unique URL that tells your browser exactly where to find it.

A URL has several parts, as shown in this example — https://blog.hubspot.com/marketing/: the scheme (https) tells the browser how to communicate with the server; the subdomain (blog) is a subdivision of the main domain; the second-level domain (hubspot) is the brand of the website; the top-level domain (.com) indicates the type or category of the website; and the subdirectory (/marketing/) points to a specific section or file on the server.

💡 Did you know? URLs were invented by Tim Berners-Lee in 1994. The longest valid URL ever recorded had over 2,000 characters long!
